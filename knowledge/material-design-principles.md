# Material Design Principles

## Quick Reference
- Material is a design system, not just a component library
- Based on tactile surfaces, bold graphic, meaningful motion
- Responsive: adapts to all screen sizes and input types
- Accessibility built into components
- Follow the documented patterns for consistency

## Deep Dive

### Core Principles
1. **Material is metaphor**: surfaces and edges provide familiar visual cues
2. **Bold, graphic, intentional**: typography, color, and imagery create hierarchy
3. **Motion provides meaning**: animation guides focus and provides continuity

### Layout System
- **Responsive grid**: 4dp baseline grid, 12-column layout
- **Breakpoints**: phone (0-599dp), tablet (600-839dp), desktop (840+dp)
- **Elevation**: z-depth indicates hierarchy (0-24dp)

### Color
- Primary, secondary, and tertiary colors
- Surface, background, and error colors
- On-colors for text/icons on colored surfaces
- Support for light and dark themes

### Typography
- Type scale: headline, title, body, label
- Responsive type: adjusts size and weight per breakpoint
- Readable at all sizes

### Components
Buttons, cards, navigation, dialogs, lists, chips, data tables, forms, etc.
Use documented components for consistency and accessibility.

### Motion
- Meaningful, not decorative
- Follows physics-based curves
- Provides feedback and continuity
- Respects reduced motion preferences

# Workflow: discovery

Consulting mode. Produce scope and a proposal. No build.

## Phases and roles
1. Discovery: atlas-consultant leads, with atlas-pm and atlas-ba
2. Feasibility: atlas-architect or atlas-ent-arch for a high-level approach; atlas-cloud for cost if relevant
3. Proposal: atlas-consultant assembles scope, options, and a SOW outline
4. Gate: final delivery (user) reviews and decides

## Default tiers
Premium: atlas-ent-arch or atlas-architect for feasibility. Standard: atlas-consultant, atlas-pm, atlas-ba.

## Definition of Done
- Problem framed (MECE); current and desired state captured.
- Options with trade-offs and rough cost.
- Proposal or SOW outline ready for the user. No build started.
